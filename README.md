# ReCord

![](https://img.shields.io/github/package-json/v/InfluenceDevs/ReCord?style=for-the-badge&logo=github&logoColor=d3869b&label=&color=1d2021&labelColor=282828)

The blurple Discord client mod forked from Vencord.

Created by Rloxx.

![](https://github.com/user-attachments/assets/3fac98c0-c411-4d2a-97a3-13b7da8687a2)

## Features

- Easy to install
- [100+ built in plugins](https://vencord.dev/plugins)
- Custom user plugin support via `src/userplugins`
- Fairly lightweight despite the many inbuilt plugins
- Excellent Browser Support: Run ReCord in your Browser via extension or UserScript
- Works on any Discord branch: Stable, Canary or PTB all work
- Custom CSS and Themes: Inbuilt CSS editor with support to import any CSS files (including BetterDiscord themes)
- Privacy friendly: blocks Discord analytics & crash reporting out of the box and has no telemetry
- Maintained very actively, broken plugins are usually fixed within 12 hours
- Settings sync: Keep your plugins and their settings synchronised between devices / apps (optional)

## Setup

1. `pnpm install`
2. `pnpm build` or `pnpm dev`
3. `pnpm inject` to patch your local Discord install

Setup config and paths are documented in `record.setup.json` and `docs/ReCord-Customization.md`.

<summary>Using Vencord violates Discord's terms of service</summary>

Client modifications are against Discord’s Terms of Service.

However, Discord is pretty indifferent about them and there are no known cases of users getting banned for using client mods! So you should generally be fine as long as you don’t use any plugins that implement abusive behaviour. But no worries, all inbuilt plugins are safe to use!

Regardless, if your account is very important to you and it getting disabled would be a disaster for you, you should probably not use any client mods (not exclusive to ReCord), just to be safe

Additionally, make sure not to post screenshots with ReCord in a server where you might get banned for it
